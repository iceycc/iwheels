## 创建一个函数

lib/utils.js

```js
'use strict';
const utils = {
    /**
     * 字符串转驼峰
     * @param str 需要转的字符串
     * @param sep 分割符，默认 '-'
     * @returns {*}
     */
    doCamel(str, sep = '-') {
        let reg = `([^${sep}])(?:${sep}+([^${sep}]))`
        return str.replace(new RegExp(reg, 'g'), function ($0, $1, $2) {
            return $1 + $2.toUpperCase();
        });
    }
}
module.exports = utils;
```



## 给所有模块添加该模块

`lerna add @iwheels/utils`
